## ✦ Tentang Repositori Ini

Repositori ini bukan tentang kode.

Ini adalah tentang seseorang yang memilih untuk mengabadikan sebuah perasaan — bukan di buku harian, bukan di surat kertas yang dilipat rapi — melainkan di dalam sebuah repository, di antara file HTML, Python, dan Jupyter Notebook.

Karena kadang, yang paling berharga dari sebuah perasaan bukanlah apakah perasaan itu dibalas atau tidak. Melainkan fakta bahwa perasaan itu **pernah ada**.

---

## ✦ Isi Repositori

```
dear-my-love/
│
├── index.html              ← Website modern, gelap, & elegan
├── nasssh.py               ← Script Python dengan animasi terminal
└── untu seseorang.ipynb    ← Jupyter Notebook interaktif
```

<br/>

### 🌐 `index.html` — Website

Website satu halaman (*single-page*) dengan desain modern dan estetik.

**Fitur:**
- Background malam gelap dengan partikel bintang animasi (Canvas API)
- Tipografi premium: **Playfair Display** × **EB Garamond** × **Inter**
- Heartbeat SVG line yang ter-*draw* saat halaman dibuka
- Scroll-reveal animation menggunakan `IntersectionObserver`
- Quote blocks & section dividers yang elegan
- Film grain noise overlay — tekstur sinematik
- Fully responsive (mobile-friendly)

**Cara buka:**
```
Buka index.html di browser mana saja
```

---

### 🐍 `nasssh.py` — Python Script

Script Python terminal yang menampilkan surat lengkap dengan animasi kaya warna.

**Fitur:**
- Auto-install dependency (`rich`) jika belum tersedia
- Loading spinner animasi saat pertama dijalankan
- ASCII art nama di intro
- **Typewriter effect** — teks muncul karakter per karakter
- Nama reveal pelan-pelan dengan warna gold-rose
- Quote blocks, section dividers, metadata panel
- Puisi penutup dengan reveal bertahap
- Final panel elegan sebagai penutup
- `Ctrl+C` graceful exit

**Cara run:**
```bash
python nasssh.py
```

> Atau paste langsung ke [Replit](https://replit.com), [OnlinePython](https://www.online-python.com/), [Programiz](https://www.programiz.com/python-programming/online-compiler), atau [OnlineGDB](https://www.onlinegdb.com/online_python_compiler).

**Dependency:**
```
rich >= 10.0
```
*(diinstall otomatis jika belum ada)*

---

### 📓 `untu seseorang.ipynb` — Jupyter Notebook

Notebook interaktif dengan tampilan HTML/CSS yang dirender langsung di dalam sel output.

**Fitur:**
- Bintang-bintang animasi (Canvas + JavaScript)
- Nama dengan gradient emas-mawar + efek glowing
- Heartbeat SVG line animasi
- Surat lengkap dengan layout elegan
- Floating hearts animasi
- Blok puisi sebagai penutup

**Cara buka:**
```bash
# Lokal
jupyter notebook "untu seseorang.ipynb"

# Atau upload ke Google Colab / Kaggle / Binder
```
